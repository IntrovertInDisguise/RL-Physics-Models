Reinforcement Learning for Physics Models

Vignesh Ramakrishnan

This repository contains implementations of reinforcement learning algorithms for physics models. The goal is to optimize a given physics-model based task by interacting with the environment.

Q-learning Algorithm

The Q-learning algorithm is based on the following update rule:

$$ Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right] $$

Where:

  • \( Q(s, a) \): Q-value for state \( s \) and action \( a \)
  • \( \alpha \): Learning rate
  • \( \gamma \): Discount factor
  • \( r \): Reward
  • \( s', a' \): Next state and action
